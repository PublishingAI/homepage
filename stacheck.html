Case studies - Stacheck tool

Statcheck is a program that checks for errors in statistical reporting in American-Psychological-Association-formatted documents. It was originally written in the R programming language. Statcheck/web is a web-based implementation of statcheck. Using statcheck/web, you can check any PDF for statistical errors without installing the R programming language on your computer. To use statcheck/web, simply upload a PDF or HTML file of your APA-formatted document. The output will identify any errors in statistical reporting from the manuscript text, based on apparent discrepancies among degrees of freedom, test statistic, and p-value.

Stacheck has been widedly applied in detecting and eliminating statistical reporting inconsistencies. It is encouraged that authors should use it before submission as more journals are using this tool in the initial triage of manuscripts and the process of peer review.
